# دریافت‌کننده قیمت و مفسر داده

دریافت داده کندل‌استیک از صرافی تووبیت و تبدیل آن به فرمت قابل استفاده برای ربات معاملاتی SMA 20.

## فایل‌ها

| فایل | توضیح |
|------|-------|
| `price.py` | دریافت داده خام کندل‌ها از API تووبیت |
| `decode.py` | تبدیل JSON خام به آبجکت‌های کندل با نام فیلدها |

## مراحل اجرا

```
price.py → {تاریخ-شمسی}.json → decode.py → candles.json → ربات SMA 20
```

## نحوه استفاده

### مرحله ۱: دریافت داده

```bash
python price.py
```

از شما خواسته می‌شود:
- **نماد ارز** — مثلاً `BTCUSDT`، `ETHUSDT`، `SOLUSDT`
- **بازه زمانی** — مثلاً `1m`، `5m`، `15m`، `1h`، `4h`
- **تعداد کندل** — مثلاً `100`، `150`، `500`

خروجی: `{تاریخ-شمسی}.json` (داده خام کندل‌ها)

### مرحله ۲: تفسیر داده

```bash
python decode.py
```

- به صورت خودکار فایل JSON را پیدا می‌کند
- اگر چند فایل باشد، از شما انتخاب می‌خواهد
- می‌توانید نام فایل را به عنوان آرگومان بدهید: `python decode.py 1404-04-01.json`

خروجی: `candles.json` (داده تفسیر شده کندل‌ها)

### مرحله ۳: اجرای ربات

```bash
python ../"bot/sma 20/main.py" candles.json
```

## پیش‌نیازها

- پایتون 3.8+
- `jdatetime` — `pip install jdatetime`
- CLI تووبیت — باید نصب باشد و در مسیر PATH قرار داشته باشد

## فیلدهای کندل

هر کندل تفسیر شده شامل این فیلدهاست:

| فیلد | توضیح |
|------|-------|
| `open_time` | زمان باز شدن کندل (میلی‌ثانیه) |
| `open` | قیمت باز شدن |
| `high` | بالاترین قیمت |
| `low` | پایین‌ترین قیمت |
| `close` | قیمت بسته شدن |
| `volume` | حجم دارایی پایه |
| `quote_volume` | حجم دارایی مظنه |
| `trades` | تعداد معاملات |
| `taker_buy_volume` | حجم خرید تیکر |
| `taker_buy_quote_volume` | حجم خرید مظنه تیکر |
