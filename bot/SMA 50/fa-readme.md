# ربات معاملاتی SMA 50 نسخه ۲.۰

موتور تحلیل کامل سیگنال‌های معاملاتی اسپات و فیوچرز (لانگ/شورت) با ۲۷ ماژول تحلیلی.

## شروع سریع

```bash
python main.py <مسیر-فایل-candles.json>
```

مثال:
```bash
python main.py ../../candles/candles.json
```

## پیش‌نیازها

```bash
pip install numpy pandas matplotlib mplfinance jdatetime
```

| فایل | توضیح |

## انواع سیگنال

| سیگنال | معنی |
|--------|------|
| STRONG BUY | خرید با اعتماد بالا |
| BUY | خرید استاندارد |
| HOLD | بدون اقدام (صبر) |
| SELL | فروش استاندارد |
| STRONG SELL | فروش با اعتماد بالا |

## امتیاز اعتماد

امتیاز ۰ تا ۱۱۵ بر اساس:
- قدرت روند (۰-۱۰)
- فاز بازار (۰-۱۰)
- هم‌راستایی اندیکاتورها (۰-۱۰)
- هم‌راستایی بازه‌ها (۰-۱۰)
- رژیم بازار (۰-۱۰)
- موقعیت فیبوناچی (۰-۵)
- ساختار موجی (۰-۱۰)
- جریان سفارش (۰-۵)
- معیارهای ریسک (۰-۵)

## ویژگی‌های نمودار

### پنل چپ — نمودار SMC
- سه‌گانه سوپرترند (ضرایب ۱.۰, ۲.۰, ۳.۰)
- برچسب‌های BOS / CHoCH
- بلوک‌های سفارش (ناحیه‌های آبی/قرمز)
- شکاف‌های ارزش منصفانه (ناحیه‌های فیروزه‌ای/قرمز)
- نشانگرهای پیوت بالا/پایین
- میله‌های حجم
- زیرپنل RSI

### پنل راست — طرح معامله
- ۱۰ کندل آخر زوم‌شده
- خطوط Entry / SL / TP1 / TP2 / TP3
- سطوح حمایت / مقاومت
- نشان خرید/فروش
- جعبه جزئیات معامله

### بخش‌های داده
- اندیکاتورهای تکنیکال (۱۷ مقدار)
- حمایت / مقاومت (R1-R5, S1-S5)
- دلایل سیگنال (رنگ‌بندی شده)
- امتیاز اعتماد (جزئیات عوامل)
- تحلیل چندبازه‌ای (۱H, ۴H, ۱D)
- سطوح فیبوناچی
- نوسان و ریسک
- رژیم بازار
- نتایج بک‌تست
- دستورالعمل‌های مدیریت ریسک
